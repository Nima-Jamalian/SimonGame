Simon Game

Start
Rule

1- Press the Start button. Simon will give the first signal. Repeat the signal by pressing the same color button.

2- Simon will give another signal. Repeat the first and the second signals by pressing the same color buttons, in order.

3- Each round Simon will add a new signal and you need to memories the pattern from the beginning sequence and repeat it.

4- Continue playing as long as you can repeat the sequence of signals correctly.
